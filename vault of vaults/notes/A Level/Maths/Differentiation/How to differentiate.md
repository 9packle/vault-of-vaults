## Power rule

When $y = x^n$, $\frac{dy}{dx} = nx^{n-1}$

## Differentiating functions

$\frac{d}{dx}(x^a+x^b) = \frac{d}{dx}(x^a) + \frac{d}{dx}(x^b)$

## Chain Rule

If $y = f(u)$ and $u = g(x)$, $\frac{dy}{dx} = \frac{dy}{du}\times\frac{du}{dx}$

e.g. for $y = a(x^2+5x+12)^2$, $\frac{dy}{dx}=2a(x^2+5x+12)\times(2x+5)$, which can be rewritten as $2a(x^3+15x^2+34x+60)$.

## Product Rule

If $y = uv$, $\frac{dy}{dx} = u\hspace{1mm}\frac{dv}{dx} + v\hspace{1mm}\frac{du}{dx}$

## Quotient Rule

If $y = \frac{u}{v}$, $\frac{dy}{dx} = \frac{v\frac{du}{dx}-u\frac{dv}{dx}}{v^2}$
